# Photography-Studio_Booking-Website

## Project Overview

**Moonshot Studio** is a modern, responsive photography studio booking platform that allows users to explore studios, view categories and packages, and book their preferred studio seamlessly. The website emphasizes a professional UI with blended hero sections, transparent headers, and modal galleries, providing a visually engaging experience.

The platform includes a smooth booking flow with confirmation and a dedicated **Admin Dashboard** for managing reservations efficiently. Administrators can view, approve, reject, and track bookings, ensuring smooth management of studio operations.

---

## Features

### User Interface
- Browse photography studios with curated categories and packages  
- Hero section with blended background images and transparent header  
- Studio cards with modals for detailed information  
- Smooth anchor-scroll navigation  
- Fully responsive design for all devices  

### Admin Dashboard
- View all bookings with detailed information  
- Search bookings by customer name  
- Approve or reject booking requests  
- Manage studio availability efficiently  

### Additional Features
- Interactive booking flow with confirmation  
- Modal popups for gallery and studio details  
- Consistent color palette: #713600, #C05800, #FDFBD4, #38240D
